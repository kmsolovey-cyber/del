# Chromatic Threshold App — Хроматический анализ изображения
Приложение для конвертации изображения в цветовое пространство YCrCb, выделения красного хроматического канала Cr по порогу и отображения полученной бинарной маски с возможностью сохранения результата.

## Возможности
Загрузка изображения.

Конвертация BGR → YCrCb (встроенная функция OpenCV).

Разбиение изображения на каналы Y, Cr, Cb.

Пороговая бинаризация канала Cr по значению ползунка (0–255).

Предпросмотр маски на вкладке «Результат».

Сохранение маски в PNG.

Удобный интерфейс с вкладками (GUI).

## Установка зависимостей
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Запуск
```bash
python task4_complex_app.py
```
## Использование
1. Перейдите на вкладку Контроль.
2. Загрузите изображение (файл image.jpg должен лежать рядом с программой).
3. Установите порог хроматического канала Cr с помощью ползунка.
4. Нажмите Выполнить — маска появится на вкладке «Результат».
5. При необходимости нажмите Сохранить маску, чтобы записать результат в PNG.
